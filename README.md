# Supermarket Sales Analytics & Forecasting
An end-to-end data pipeline and interactive dashboard built for retail demand analysis.

### Features
* **Exploratory Data Analysis (EDA):** Dynamic filtering using Pandas to uncover sales trends.
* **Interactive Visualizations:** Plotly charts mapping revenue by product category and payment method.
* **AI Demand Forecasting:** A Scikit-Learn Linear Regression model projecting 7-day future sales to optimize inventory management.

### Tech Stack
Python, ML, Streamlit, Pandas, Plotly, Scikit-Learn
